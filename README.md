# SMS Agent

A Python-based SMS agent application for handling SMS communications.

## Features

- SMS message processing
- Agent-based architecture
- Configurable settings

## Installation

```bash
pip install -r requirements.txt
```

## Usage

```bash
python run.py
```

## Configuration

Update configuration settings in the config file before running the application.

## Requirements

- Python 3.7+
- Required dependencies listed in requirements.txt
